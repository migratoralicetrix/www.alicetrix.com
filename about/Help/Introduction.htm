Introduction

Alicetrix tries to bring to you Enterprise Digital Workplace Migrator, a web application based Workplace Migrator for migrating your contents from various sources that host Mails, Files or Calendars. 

Currently available offers are:

  • Google Workspace GMAIL: Migrate mails and  forward calendars (only forwards future calendar) from Gmail Current Software Offerings are:
    • Gmail To Exchange Online
    • Gmail to Gmail
  • EXCHANGE Online: Migrate emails and forward calendars (only forwards future calendar) from Exchange Online Current Software Offerings are:
    • Exchange Online to Gmail
    • Exchange Online to Exchange Online
  • IMAP Mailboxes: Migrate two and from any IMAP, Gmail, Exchange Online accounts accounts. 
    • IMAP To/From Gmail
    • IMAP To/From Exchange Online
    • IMAP To/From IMAP
  • Google Drive: Migrate contents from Google Drive, Create new folders, Migrate permissions, Get file content reports from Google Drive, and convert your Google forms, sheets, and docs into Microsoft Office documents.Current Software Offerings are:
    • Google Drive to OneDrive
    • Google Drive to SharePoint Online
  • BOX: Migrate contents from Box, Create new folders, Migrate permissions, Get file content reports from Box Current Software Offerings are:
    • Box To OneDrive
    • Box to SharePoint Online
  • OneDrive for Business: Migrate contents, Create new folders, Migrate permissions, Get file content reports. Current Software Offerings are:
    • OneDrive To OneDrive
    • OneDrive To SharePoint Online
  • SharePoint Online: Migrate (only) document library, Create new folders, Migrate permissions, Get file content reports. Current Software Offerings are:
    • SharePoint Online To SharePoint Online
    • SharePoint Online To OneDrive

Below terminology will be used frequently in help file and application.

  • Source - Tenant from where User contents will be migrated
  • Destination - Tenant where User contents will be migrated

Key Features of Alicetrix Workplace Migrator are:

  • Multi-Threaded Environment – Contents(emails/files) of single users are uploaded read and uploaded into destination tenant through a multi-threaded program enabling simultaneous downloads/uploads hence fast migration.  During large file migration, larger files are broken into small pieces for upload, and for each small segment of files multithreading is also done. 
  • Starts Migrating at once and, does not create a repository of contents before migration. It builds repository while doing migration useful in case of very large data or lot of files in millions.
  • Resync-Migration Options: Multiple Migration Options are available for migrations that is first-time migration, resyncing of migration (local respository is maintianed to ensure syncing is as fast as possible).
  • Permission Migration – Ability to migrate permission using Various algorithms are available.
  • Managed failovers - Complex algorithm built to ensure failures are handled in most case automatically by the application through retrials, wait, requeuing of file transfers or auto-correction in certain cases.
  • Security with your own Cloud Infrastructure – Migrator relies on secured Microsoft Azure Infrastructure or your infrastructure for migration. Individual instance of this application are deployed through Azure Marketplace into customer’s tenant’s/environment. This ensures advanced security and data encapsulation which is sometimes needed for highly secure enterprise.
  • Remediation: It allows to generate reports on Source Tenant. In case of File Migration, it also corrects file names if users opt for that option, getting rid of special characters etc. (for OneDrive Destination Tenant)
  • Data Storage: Migrator relies on local database for storage of files transaction information in Web App Server itself. Local database storage ensures roundtrip in accessing database is minimized ensuring faster migration. Auto caching is enabled ensuring Database access is only done a specific period of times.
  • Auto-conversions for Google: Migrates Google Docs as well, Sheets, Images into respective file outputs.
  • No Contents Data storage in Web App: Data read is never stored on drives.
  • Complete Reports: Complete reports are made available using excel exports.
  • Complete and Intuitive Dashboard for Migration: Online updates on status of migration is available with a mobile friendly interface.
  • Admin Controls: Allows for granular admin controls on few aspects like retrials when failure, wait times etc. for more flexibility of setting up optimized speed of migration in certain cases.
